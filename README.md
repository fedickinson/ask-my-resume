# RAG Resume Assistant

A RAG (Retrieval-Augmented Generation) system for creating personalized resume assistance.

## Project Structure

```
rag-resume-assistant/
├── README.md
├── docs/
│   └── project-plan.md          ← Project planning documents
├── content/                      ← Phase 1 deliverable
│   ├── narrative/
│   ├── projects/
│   ├── ai-thinking/
│   ├── experience/
│   ├── education/
│   ├── working-style/
│   ├── vision/
│   └── practical/
├── src/                          ← Phase 2+
│   ├── embedding/
│   ├── retrieval/
│   ├── generation/
│   └── api/
├── prompts/                      ← Phase 3
│   └── system-prompt-v1.md
├── frontend/                     ← Phase 5
└── eval/                         ← Phase 4
    └── test-queries.md
```

## Development Phases

- **Phase 1**: Content collection and organization
- **Phase 2**: Embedding and retrieval system
- **Phase 3**: Prompt engineering
- **Phase 4**: Evaluation framework
- **Phase 5**: Frontend interface
